# v2 ideas — parking lot

If a temptation to add something outside v1 scope appears during build, it goes here. v1 ships first.

Out of scope for v1 per the v1 maxxed spec — do NOT build, capture ideas here:

- Marketplace / public Recipe directory / third-party recipes (closed ecosystem in v1)
- Multi-org workflows (per-team works in v1; cross-org is v2)
- On-prem / self-hosted (SaaS only in v1; self-hosted is v3 if ever)
- Windows / Linux clients (macOS only)
- Generative video / video output / recording playback
- Real-time collaborative editing of the artifact (Notion/Figma territory)
- Chat product features (don't compete with Slack)
- Calendar integration / auto-scheduling (don't compete with Granola's calendar)
- Auto-deploy without explicit user tap (v1 = explicit; v2+ may add trust-gated auto-deploy)
